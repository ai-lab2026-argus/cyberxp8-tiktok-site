TikTok draft uploader workflow

This site supports the CYBERXP8 TikTok developer/app setup by providing public website, privacy, and terms pages. The uploader itself is a private/internal utility for the account owner.

Important: CYBERXP8 Draft Uploader should only use TikTok OAuth and approved API workflows. It does not ask for TikTok passwords, and when configured with video.upload, it prepares uploads for manual completion rather than direct publishing.
