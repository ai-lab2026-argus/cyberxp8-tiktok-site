# CYBERXP8 TikTok Site

Static website for the CYBERXP8 TikTok/developer-app effort.

## Files

- `index.html` — simple public project website
- `privacy.html` — Privacy Policy for CYBERXP8 Draft Uploader
- `terms.html` — Terms of Service for CYBERXP8 Draft Uploader
- `styles.css` — shared styling
- `assets/cyberxp8-icon.png` — site icon/logo

## Upload to GitHub Pages

1. Create a new public GitHub repository, for example `cyberxp8-tiktok-site`.
2. Upload all files in this folder to the repository root.
3. In GitHub, go to **Settings → Pages**.
4. Under **Build and deployment**, choose **Deploy from a branch**.
5. Select branch **main** and folder **/** root, then save.
6. Your website URL will be similar to:

   `https://YOUR-GITHUB-USERNAME.github.io/cyberxp8-tiktok-site/`

Use that URL as the app website. Use these direct URLs for policy fields:

- Privacy Policy: `https://YOUR-GITHUB-USERNAME.github.io/cyberxp8-tiktok-site/privacy.html`
- Terms of Service: `https://YOUR-GITHUB-USERNAME.github.io/cyberxp8-tiktok-site/terms.html`
